## 摘要
现实世界中的多模态系统必须对低质量数据具有鲁棒性，例如传感器噪声、不完整的多模态数据和冲突输入。然而，现有的可信融合方法依赖于模型自身的预测置信度来判断数据质量。这形成了一种循环依赖：当模型自信但错误时，这些方法无法检测到错误。为了打破这种循环，我们提出了基于几何的多模态融合（GMF）。GMF不依赖预测，而是通过测量输入在潜在空间中需要多少传输校正来评估可靠性。我们使用整流流实现扩散薛定谔桥传输，其中初始速度的平方给出了一个高效的学习校正分数。有效数据具有较低的平方速度幅值，而噪声数据、不完整数据或冲突数据需要更强的传输校正。这种基于几何的可靠性信号充当独立判断，即使在分类器被欺骗时也能有效标记不可靠输入。大量实验表明，与基于置信度的基线方法相比，GMF显著提升了针对严重传感器噪声和语义冲突的鲁棒性。

## Abstract
Real-world multimodal systems must be robust against low-quality data, such as sensor noise, incomplete multimodal data and conflicting inputs. However, existing trustworthy fusion methods rely on the model's own prediction confidence to judge data quality. This creates a circular dependency: when a model is confident but wrong, these methods fail to detect the error. To break this loop, we propose Geometry-based Multimodal Fusion (GMF). Instead of relying on predictions, we evaluate reliability by measuring how much transport correction the input needs in latent space. We implement Diffusion Schrödinger Bridge transport with Rectified Flow, where the squared initial velocity gives an efficient learned correction score. Valid data has low squared velocity magnitude, while noisy, incomplete data or conflicting data requires stronger transport correction. This geometry-based reliability signal acts as an independent judge, effectively flagging unreliable inputs even when the classifier is fooled. Extensive experiments demonstrate that GMF significantly improves robustness against severe sensor noise and semantic conflicts compared to confidence-based baselines.

## 2. 论文提出的方法论
- **核心思想**：将可靠数据视为遵循固有传输模式的样本，利用**扩散薛定谔桥（Diffusion Schrödinger Bridge, DSB）** 建模潜在分布间的传输，并以**整流流（Rectified Flow）** 将其线性化，快速估计传输代价。
- **关键技术细节**：
  - **内模态传输成本**：对每个模态 \(m\) 的特征 \(\mathbf{z}^{(m)}\)，计算其向先验参考分布传输的初始速度平方 \(\|\mathbf{v}^{(m)}_\theta(\mathbf{z}^{(m)},0)\|^2\)，作为几何可靠性得分。干净数据距离流形近，成本低；噪声或异常数据成本高。
  - **跨模态传输成本**：对模态对 \((a,b)\)，通过方向特异性跨模态速度场 \(\mathbf{v}^{(a\to b)}_\Phi\) 进行一步映射，计算映射后特征与目标模态特征的欧氏距离 \(\|\Phi_{a\to b}(\mathbf{z}^{(a)})-\mathbf{z}^{(b)}\|^2\)，捕捉语义冲突。
  - **融合权重生成**：利用内模态成本通过玻尔兹曼分布产生竞争基分 \(\beta^{(m)}_{\text{comp}}\)；利用跨模态成本生成交互门 \(\gamma^{(m)}_{\text{int}}\)，聚合来自可靠邻居的共识，最终得到归一化权重 \(w^{(m)}\)，并调制证据网络的参数 \(\alpha\)。
  - **训练目标**：联合优化几何传输损失（整流流匹配损失）、证据任务损失（期望交叉熵+KL正则）和冲突感知正则项（全局一致性系数引导的均匀化惩罚），并通过**梯度分离**策略防止几何与决策分支相互干扰。
- **理论保证**：在潜在流形集中性和可分离性假设下，证明了融合权重为熵正则化几何成本函数的全局最小解，并给出了语义冲突下跨模态成本的几何屏障下界，从而冲突模态的权重被指数级抑制。

## 8. 不足与局限
- **对特征提取器的依赖**：方法完全建立在已有编码器的潜在表示之上，若上游特征未能形成可分流形（如模态坍塌），几何成本将失效，此时模型只会输出均匀分布，无法恢复语义。

- **假设严格**：理论保证依赖于潜在空间流形的集中性和可分性（且要求跨模态映射语义一致），现实中不一定完美满足。
