## 摘要
近期生成建模的进展通过利用受流匹配启发的部分免模拟训练方法，实现了高维设置下薛定谔桥（SB）的高效计算。然而，这些方法尚未涵盖具有反射动力学的SB，这是一种有用的模型选择，具有内置保证，确保生成的样本保持在数据域内。现有的反射SB替代方法则依赖于基于前向-后向SDE理论的更复杂训练，需要昂贵的高阶导数并在训练期间采样整个路径。在本文中，我们引入了一个部分免模拟框架，使得反射SB可以像流匹配一样进行训练，使用新的采样方法和回归目标。我们通过耦合已知的高维图像数据集对来展示我们的结果。使用反射动力学会在训练和推理过程中产生可忽略的额外挂钟时间，同时保持或略微提升生成性能。

## Abstract
Recent advances in generative modeling have enabled the efficient computation of Schrödinger bridges (SB) in high-dimensional settings by leveraging partially simulation-free training methods inspired by flow matching. However, these have not covered SBs with reflecting dynamics, a useful model choice with built-in guarantees that generated samples stay in the data domain. Existing alternatives for reflected SBs instead rely on more complex training based on forward--backward SDE theory, requiring expensive higher-order derivatives and sampling entire paths during training. In this article, we introduce a partially simulation-free framework that allows reflected SBs to be trained similarly to flow matching, using a new sampling method and regression target. We demonstrate our results by coupling pairs of well-known high-dimensional image datasets. Using reflected dynamics incurs negligible additional wall-clock time during both training and inference while maintaining or slightly improving generative performance.

## 2. 论文提出的方法论

### 核心思想
- 将迭代马尔可夫拟合（IMF）算法扩展到反射参考过程。使用反射布朗运动（RBM）作为参考过程，确保生成样本位于单位超立方体 \([0,1]^d\) 内。
- 利用 Doob’s h-变换和 Girsanov 定理推导反射过程下的漂移修正项，使反射桥的采样和得分函数计算变得高效。

### 关键技术细节
1. **参考过程**：反射布朗运动
   \[
   dX_t = \sigma dW_t + n(X_t) dL_t,
   \]
   其中 \(n(x)\) 是内法向量，\(L_t\) 是边界局部时。
2. **马尔可夫投影**：得到反射 SDE 的漂移修正
   \[
   v^*(t,x) = \sigma^2(t) \, \mathbb{E}_{\Pi_{1|t}} \bigl[ \nabla \log Q^r_{1|t}(X_1 \mid X_t) \mid X_t = x \bigr],
   \]
   其中 \(Q^r_{1|t}\) 是反射过程的转移密度。
3. **目标函数**：最小化 KL 散度，得到回归损失（含前向和反向两个方向）
   \[
   L_{\to}(\theta;\Pi) = \int_0^1 \frac{1}{\sigma^2(t)} \mathbb{E}_{\Pi_{t,1}} \bigl[ \lVert \sigma^2(t) \nabla \log Q^r_{1|t}(X_1 \mid X_t) - v_\theta(t,X_t;\to) \rVert^2 \bigr] dt,
   \]
   反向类似。
4. **反射桥的高效采样**（算法 2）：利用反射操作 \(R\) 和因式分解成分维独立，实现近似桥梁采样。
   - 对每个维度独立计算转移密度的高斯和，截断到有限反射次数（实验中取 11 次）。
5. **α-RSBM 算法**：采用在线训练，每步仅执行一次梯度更新，同时优化前向和后向方向。

### 算法流程（α-RSBM）
- **预训练阶段**：从独立乘积 \((π_0 ⊗ π_1)\) 采样端点，采样桥梁中点 \(X_t \mid X_0, X_1\)，优化损失。
- **微调阶段**：从当前模型采样新的终点 \(\hat{X}_1 \mid X_0\)，然后采样桥梁 \(X_t \mid X_0, \hat{X}_1\)；反向也做类似操作。梯度步更新网络，用 EMA 权重改善结果。

- 采用 EMA 模型（衰减率 0.999）用于采样。

## 8. 不足与局限
