## 摘要
标准扩散模型通常使用单一时间齐次的高斯终端分布作为生成的参考律。虽然这一选择在分析上方便且在经验上有效，但对于集中在低维流形附近的数据，它几乎没有提供明确的结构，而数据分布的不同区域可能对应于不同的局部几何或语义因素。因此，反向模型必须几乎完全从无结构的终端参考分布中恢复流形级别的结构。我们提出PTL-扩散，这是一种概念验证的扩散框架，其前向加噪过程收敛到一个非常数的周期高斯终端定律族，而不是单一的不变定律。与相位条件DDPM不同（其中相位信息仅进入去噪网络，而前向过程保持不变），PTL-扩散将相位结构直接嵌入到前向加噪动力学中。所提出的构造与标准去噪扩散模型保持接近：对于周期强迫的奥恩斯坦-乌伦贝克型前向过程，我们推导出闭式的前向边缘分布、极限周期高斯终端族以及显式的高斯反向后验，从而实现标准的噪声预测训练。我们还引入了一个不变平均正则化项，通过平均周期参考律耦合相位条件反向动力学。在环面和圆柱点云基准以及Olivetti人脸数据集上的实验表明，PTL-扩散在流形级别上的分布匹配优于匹配的DDPM基线，减少了相位条件误差、特征空间协方差误差和最近邻流形距离。这些结果表明结构化的终端参考定律是一个有前景的方向，同时激励了更具表现力的相位构造和更大规模的评估。

## Abstract
Standard diffusion models typically use a single time-homogeneous Gaussian terminal distribution as the reference law for generation. While this choice is analytically convenient and empirically powerful, it provides little explicit structure for data concentrated near low-dimensional manifolds, where different regions of the data distribution may correspond to distinct local geometric or semantic factors. As a result, the reverse model must recover manifold-level structure almost entirely from an unstructured terminal reference distribution.   We propose PTL-Diffusion, a proof-of-concept diffusion framework whose forward noising process converges to a nonconstant periodic family of Gaussian terminal laws rather than to a single invariant law. Unlike a phase-conditioned DDPM, where phase information only enters the denoising network while the forward process remains unchanged, PTL-Diffusion embeds phase structure directly into the forward noising dynamics.   The proposed construction remains close to standard denoising diffusion models: for a periodically forced Ornstein--Uhlenbeck-type forward process, we derive closed-form forward marginals, the limiting periodic Gaussian terminal family, and explicit Gaussian reverse posteriors, enabling standard noise-prediction training. We also introduce an invariant-average regularization term coupling the phase-conditioned reverse dynamics through the averaged periodic reference law. Experiments on torus and cylinder point-cloud benchmarks and the Olivetti face dataset show that PTL-Diffusion improves manifold-level distributional matching over matched DDPM baselines, reducing phase-conditioned errors, feature-space covariance errors, and nearest-neighbour manifold distances. These results suggest structured terminal reference laws as a promising direction, while motivating more expressive phase constructions and larger-scale evaluations.

## 2. 论文提出的方法论

### 核心思想
- 设计一个前向加噪过程，其长期行为不是趋向于单一不变分布，而是趋向于一个 \(P\) 周期的高斯终端族 \(\{ \mu^*_r \}_{r=0}^{P-1}\)，其中相位变量 \(r\) 作为粗粒度坐标，用于组织数据分布的变化。
- 与仅对去噪网络进行相位条件（Phase-DDPM）不同，PTL-Diffusion 将相位结构直接嵌入前向动力学中，使终端参考分布本身具有结构。

### 关键技术细节

#### 前向动力学：周期性强迫OU过程
- 递推形式：
  \[
  x_{n+1} = \rho x_n + b_{r_0 + n} + \sigma \varepsilon_{n+1}, \quad \varepsilon_{n+1} \sim \mathcal{N}(0, I_d),
  \]
  其中 \(0<\rho<1\)，\(\sigma>0\)，\(\{b_n\}\) 是周期为 \(P\) 的非常数驱动力，相位索引模 \(P\) 理解。
- **闭式前向边缘**（引理1）：
  \[
  q(x_n \mid x_0, r_0) = \mathcal{N}\big(m_{n, r_0}(x_0), \Sigma_n\big),
  \]
  其中均值含周期性项，协方差 \(\Sigma_n = \sigma^2 \frac{1-\rho^{2n}}{1-\rho^2} I_d\)。
- **极限周期族**（定理1）：
  \[
  \mu^*_r = \mathcal{N}(m^*_r, \Sigma^*), \quad m^*_r = \sum_{j=0}^\infty \rho^j b_{r-1-j},\quad \Sigma^* = \frac{\sigma^2}{1-\rho^2} I_d.
  \]
  前向一步将 \(\mu^*_r\) 映射到 \(\mu^*_{r+1}\)，且对于任意初始分布，Wasserstein-1 距离指数衰减到对应相位成员。

#### 反向过程
- 高斯反向后验（引理2）：
  \[
  q(x_n \mid x_{n+1}, x_0, r_0) = \mathcal{N}\big(\tilde \mu_{n, r_0}(x_{n+1}, x_0), \tilde \Sigma_n\big).
  \]
  噪声参数化形式可写为 \(\tilde \mu = \frac{1}{\rho}(x_{n+1} - b_{r_0+n}) - \kappa_n \varepsilon\)，从而可使用噪声预测网络。
- 相位编码：用正弦/余弦嵌入 \(\phi(r) = (\sin\frac{2\pi r}{P}, \cos\frac{2\pi r}{P})\)。

#### 不变平均正则化
- 引理3：在提升的相-状态空间 \(\{0,\dots,P-1\}\times\mathbb{R}^d\) 上，\(\Pi^* = \frac{1}{P}\sum_r \delta_r \otimes \mu^*_r\) 是不变测度，其边缘 \(\bar\mu^* = \frac{1}{P}\sum_r \mu^*_r\) 称为不变平均参考律。
- 正则化项（公式6）：
  \[
  L_{\text{avg}} = \mathbb{E}_{x_0,\varepsilon,n,r_0}\left[\left\| \frac{1}{P}\sum_{s=0}^{P-1}\varepsilon_\theta(x_n, n, \phi(s)) - \varepsilon \right\|^2\right],
  \]
  旨在耦合各相位的去噪网络，使之平均行为符合平均参考律。
- 完整训练目标：\(L = L_{\text{phase}} + \lambda L_{\text{avg}}\)，其中 \(L_{\text{phase}} = \mathbb{E}\left[\|\varepsilon - \varepsilon_\theta(x_n, n, \phi(r_n))\|^2\right]\)。

#### 数据自适应周期驱动力
- 给定训练样本及其相位标签，计算相位中心 \(c_r\)，然后中心化 \(\tilde c_r = c_r - \bar c\)，定义驱动力 \(b_r = \tilde c_{r+1} - \rho \tilde c_r\)，满足零平均条件 \(\frac{1}{P}\sum_r b_r = 0\)。

### 算法流程（文字说明）
- **训练（算法1）**：为每个样本及其相位 \(r_0\)，随机采样扩散步 \(k\) 和噪声 \(\varepsilon\)，利用闭式边缘直接生成 \(x_k\)，计算当前相位 \(r_k = r_0 + k \mod P\)，返回 \((x_k, r_k, \varepsilon)\) 用于训练噪声预测网络。
- **生成（算法2）**：采样初始相位 \(r_0\)，从有限步终端近似采样 \(x_K\)，然后从 \(k=K-1\) 到 0 反向迭代：预测噪声 \(\hat \varepsilon = \varepsilon_\theta(x_{k+1}, k+1, \phi(r_{k+1}))\)，计算反向均值，再采样 \(x_k\)，最终输出 \(x_0\)。

## 3. 实验设计

### 数据集 / 场景
1. **合成环面点云（Torus）**：嵌入 \(\mathbb{R}^4\)，两个角度满足 \(\theta^{(2)} = 2\theta^{(1)} + \xi \mod 2\pi\)，加小噪声；相位由 \(\theta^{(1)}\) 分箱得到。
2. **合成圆柱点云（Cylinder）**：嵌入 \(\mathbb{R}^3\)，角度和高度均匀采样；相位由角度分箱得到。
3. **Olivetti 人脸数据集**：400 张 64×64 灰度人脸图，PCA 降维取前两主成分，相位由 atan2 角度分箱得到。

### 基准方法
- **DDPM**：标准扩散模型，单一高斯终端。

- **PTL-Diffusion (w/o reg.)**：使用周期性前向和相位嵌入，但不加不变平均正则化。
- **PTL-Diffusion (full)**：完整模型。

所有方法使用相同的去噪网络架构和训练配置（合成数据 60 epoch，人脸数据 5000 epoch）。

## 6. 论文的主要结论与发现

- PTL-Diffusion 在合成环面和圆柱点云上，显著降低了流形约束误差、相位条件误差和不变平均误差，优于 DDPM 和 Phase-DDPM。
- 在人脸数据集上，PTL-Diffusion 在所有主要指标（像素级误差、PCA 空间误差、最近邻距离）上均大幅低于基线，而 Phase-DDPM 有时甚至比 DDPM 更差，说明单纯加相位条件不足，必须改变前向参考分布。
- 不变平均正则化虽然能稳定训练并略有提升，但主要增益来自周期性终端族本身。
- 敏感性分析表明，周期 \(P\) 过小会导致性能下降，但 \(P=30\) 时处于稳定区域。
