## 摘要
我们提出了一种几何上无坐标的解法，用于解决李群$\mathsf{SO}(2)$上运动学方程的向同性薛定谔桥问题（SBP）。我们将系统的角速度视为控制输入，并假设所给定的在$\mathsf{SO}(2)$上的初始和终端状态概率密度函数是连续且严格正的。我们通过证明存在唯一解，即所谓的$\mathsf{SO}(2)$上的薛定谔方程组，来求解SBP，这通过证明在希尔伯特射影度量下，一个不动点递归在完备度量空间中是压缩的来实现。这样设计的几何控制器仅利用$\mathsf{SO}(2)$的内在几何结构，而不将其嵌入到欧几里得平面中，以实现最优密度控制。数值模拟验证了薛定谔桥理论构造的有效性。代码和动画公开于\texttt{\href{https://gitlab.com/a5akhtar/sbp}{https://gitlab.com/a5akhtar/sbp}}。

## Abstract
We present a geometric coordinate-free solution to the isotropic Schrödinger bridge problem (SBP) for the kinematic equation on the Lie group $\mathsf{SO}(2)$. We consider the angular velocity of the system as the control input and assume that the given initial and terminal state probability density functions defined on $\mathsf{SO}(2)$ in our SBP are continuous and strictly positive. We solve the SBP by proving the existence and uniqueness of a solution to the so-called Schrödinger system of equations on $\mathsf{SO}(2)$, by showing that a fixed-point recursion is contractive in a complete metric space with respect to the Hilbert's projective metric. The geometric controller thus designed only uses the intrinsic geometric structure of $\mathsf{SO}(2)$ and does not embed it in the Euclidean plane to achieve the optimal density control. The numerical simulation verifies the validity of the theoretical construction of the Schrödinger bridge. The code and animations are publicly available at \texttt{\href{https://gitlab.com/a5akhtar/sbp}{https://gitlab.com/a5akhtar/sbp}}.

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将角速度 \(\Omega\) 视为控制输入，系统由 Stratonovich SDE 驱动。通过 **Hopf–Cole 变换** 将非线性的随机最优控制问题转化为一对 **线性的正向和反向热方程**，进而得到 **Schrödinger 系统** 方程组。
- **关键技术细节**：
  1. **模型**：系统状态 \(R(t) \in \mathsf{SO}(2)\)，运动学方程为  
     \[
     dR(t) = R(t)\widehat{\Omega}(R,t)\,dt + \sigma \sum_{i=1}^m R(t) B_i \circ dw_i(t),
     \]
     其中 \(\widehat{\Omega}\) 是角速度对应的李代数元素，\(\sigma>0\) 为扩散强度。
  2. **最优性条件**：通过拉格朗日方法得到耦合 PDE 系统，再经 Hopf–Cole 变换  
     \[
     \phi(R,t) = \exp\left(\frac{\Psi(R,t)}{\sigma^2}\right),\quad 
     \psi(R,t) = \rho_{\text{opt}}(R,t)\exp\left(-\frac{\Psi(R,t)}{\sigma^2}\right),
     \]
     得到 \(\partial_t \phi = -\frac{\sigma^2}{2}\Delta_{\mathsf{SO}(2)}\phi\) 和 \(\partial_t \psi = \frac{\sigma^2}{2}\Delta_{\mathsf{SO}(2)}\psi\)。
  3. **Schrödinger 系统**：边界条件耦合为  
     \[
     \rho_0 = \psi_0\, T_1 \phi_1,\quad \rho_1 = \phi_1\, T_1\psi_0,
     \]
     其中 \(T_1\) 是时间 \(1\) 的热算子，\(\phi_1 = \phi(\cdot,1),\ \psi_0 = \psi(\cdot,0)\)。
  4. **解的存在唯一性**：利用 **Hilbert 射影度量** \(d_H\) 在严格正连续函数空间 \(C_+(\mathsf{SO}(2))\) 上构造映射  
     \[
     F = R_{\rho_1} \circ T_1 \circ R_{\rho_0} \circ T_1,
     \]
     其中 \(R_{\rho}(g) = \rho/g\)。证明 \(F\) 是收缩映射（收缩因子 \(c^2<1\)），从而在单位球面上存在唯一不动点，给出 \(\phi_1\) 和 \(\psi_0\)。
  5. **控制器形式**：最终的最优控制器为  
     \[
     \Omega_{\text{opt}}(R,t) = \left( \sigma^2 R^\top \operatorname{grad} \log \phi(R,t) \right)^\vee,
     \]
     其中 \(\phi\) 通过热半群反向传播得到，表达式完全依赖 \(\mathsf{SO}(2)\) 的几何结构。
